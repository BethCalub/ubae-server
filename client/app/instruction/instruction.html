Instructions

The instruction table contains the services offered by the University and its processes. This table consists data which answers HOW questions.

  • The variable 'name' contains the name of the service.
  • The variable 'process' contains the processes involved in the service.
  • The 'message' variable contains the sentence form of 'process' which will serve as UBAEs reply. Try to make it witty as possible as to emulate more human-like thinking.
  • The variable 'tags' contains keywords that will be detected to determine which data UBAE will use as a response to the inquiry. Try to add more tags which are possible keywords.
